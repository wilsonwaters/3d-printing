# 3D Printing Repo

## Structure

- `.claude/skills/` — AI skill source files (Claude Code native format)
- `.claude-plugin/` — Plugin and marketplace manifests for distribution
- `adapters/` — Auto-generated single-file versions for other AI frameworks (do not edit manually)
- `3d-models/` — Parametric OpenSCAD models and 3D printing projects
- `scripts/` — Build tools

## Working with Skills

Skills in `.claude/skills/` are live — they work automatically when using Claude Code in this repo. Edit them directly and test by asking Claude to design parts.

The `adapters/` directory is auto-generated by `scripts/build-adapters.py` and rebuilt via GitHub Actions on push. Do not edit adapter files directly — changes will be overwritten.

## Working with Models

Each model lives in its own directory under `3d-models/` with a brief README explaining what it is.
